Linguagens que utilizo no meu dia.

Atualmente, utilizo essas três linguagens para a criação de sites e aplicações para a internet.

HTML

HyperText Markup Language ou Linguagem de Marcação de Hipertexto é uma linguagem de marcação que torna possível a construção de sites e a inclusão de elementos, como imagens e vídeos.

CSS

Cascading Style Sheets ou Folhas de Estilo em Cascata é uma linguagem utilizada para aplicar estilos a elementos de uma página da internet, escrita em uma linguagem de marcação, como o HTML.

JavaScript

É uma linguagem de programação que permite inserir elementos complexos em páginas da internet, como conteúdos que se atualizam em um intervalo de tempo, mapas interativos ou gráficos animados.
